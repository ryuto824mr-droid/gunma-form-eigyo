はじめに

LOCLE営業ツールは「企業リサーチ → フォーム/メール送信 → 反応分析」を自動化するツールです。

基本的な使い方の流れ：

  1. 企業リストに営業先企業を登録する
  2. 自動リサーチでフォームを解析する
  3. メッセージ管理でバリアント（文面テンプレート）を作成する
  4. 送信管理で企業と文面を選んで送信する
  5. 分析ページで反応率を確認して文面を改善する

企業リストページ

企業の追加方法

  • 手動追加：企業名・URL・メールアドレスを入力して「追加」ボタンをクリック。追加後に自動でリサーチが開始されます。
  • CSV一括インポート：name,url,email 形式の CSV ファイルを選択して「インポート」。1行目はヘッダー行、email 列は任意。既存 URL との重複はスキップされます。
  • 自動検索：地域とキーワードを入力して「検索」ボタンで候補企業を一覧表示し、選んでまとめて追加できます。

自動リサーチの使い方

  • 個別リサーチ：各行の「リサーチする」ボタンをクリックすると、その企業だけリサーチします。
  • 一括リサーチ：「未リサーチを一括リサーチ」ボタンで、pending 状態の企業を順番に自動解析します。

ステータスの意味

ステータス意味
未リサーチまだ解析していない初期状態
✓ 完了フォーム解析が完了した状態
🔒 CAPTCHACAPTCHAが検出され自動送信できない
フォームなしお問い合わせフォームが見つからなかった
✕ エラー解析中にエラーが発生した

自動化 OK / NG の意味

  • OK：フォームへの自動入力・送信が可能です。送信管理ページで「フォームに自動送信」が使えます。
  • NG：フォームが見つからない・CAPTCHA があるなど自動化できません。手動でコピー送信するか、メールアドレスを登録してメール送信を利用してください。

メッセージ管理ページ

バリアントの作成方法

名前・チャネル（メール or フォーム）・件名（メールの場合）・本文を入力して「作成」ボタンをクリックします。

差し込み変数 {{company_name}}

本文テンプレートに {{company_name}} と書くと、送信時に企業名が自動で入ります。例：「{{company_name}} ご担当者様」→「株式会社○○ ご担当者様」

タグの使い方

key: value 形式でタグを任意に追加できます。分析ページでタグの key/value ごとに反応率を比較できるため、A/B テストの軸として活用できます。例：industry: IT、size: 中小

チャネルの違い

  • メール：Gmail OAuth 経由でメールアドレスに直接 DM 送信します。企業にメールアドレスが登録されている場合に使えます。
  • フォーム：ヘッドレスブラウザで企業サイトのお問い合わせフォームに自動入力して送信します。自動化 OK の企業のみ利用可能です。

送信管理ページ

下書き生成の手順

  1. 「企業」セレクトで送信先企業を選ぶ（★ マークはフォーム自動化 OK の企業）
  2. 「バリアント」セレクトで使用する文面テンプレートを選ぶ
  3. 「下書き生成」ボタンをクリックするとプレビューが表示される

フォーム自動送信とメール送信の違い

  • フォームに自動送信：ヘッドレスブラウザを使って企業サイトのフォームに自動入力します。自動化 OK の企業にのみ表示されます。
  • メールで送信：Gmail OAuth 経由でメールを送信します。企業にメールアドレスが登録されている場合のみ表示されます。

強制送信チェックボックス

同一企業への 24 時間以内の重複送信をスキップするチェックを無効にします。同じ企業に再送信したい場合はオンにしてください。

反応の記録方法

送信履歴テーブルの「反応を記録」ボタンをクリックすると、分類（興味あり / 辞退 / 質問 / その他）と自由メモを入力できます。記録した反応は分析ページに反映されます。

分析ページ

バリアント別集計の見方

各バリアント（文面テンプレート）ごとに送信数・反応数・興味あり数・反応率を集計します。反応率が高いバリアントが有効な文面です。

タグ別集計の見方

バリアントに設定したタグの key/value ごとに反応率を集計します。例：業種・企業規模などの軸で A/B テストの比較ができます。

返信チェックボタンの使い方

「返信チェック」ボタンをクリックすると、Gmail の未読メールを確認して送信履歴と照合し、返信があった場合に自動で反応を記録します。毎朝 9 時にも自動実行されます。

よくある質問

フォーム自動送信が「エラー」になる

相手サイトのフォーム構造が複雑な場合があります。その場合は「コピー」ボタンで文面をコピーして手動送信するか、メールアドレスを登録してメール送信をお試しください。

CAPTCHAとは？

自動送信を防ぐセキュリティ機能です。CAPTCHA 付きのフォームは自動送信できません。手動送信またはメール送信を使用してください。

同じ企業に2回送信したい

送信管理ページの「強制送信」チェックボックスをオンにすると、24 時間制限・送信済みチェックをスキップして送信できます。

分析ページに数字が表示されない

送信管理ページで送信を記録すると集計が表示されます。「送信済みとして記録」ボタンか自動送信を実行してください。
